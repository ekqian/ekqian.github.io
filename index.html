ACVSN Checker

Developed a command line interface app which takes in suborbital field data from satellites and verifies the correctness of the data products according to the Atmospheric Composition Variable Standard Name Convention.

Created during my time at NASA and has assisted data science team in making data more findable, accessible, interoperable, and reusable (FAIR).

PropelPixel

Currently working on the backend for a mobile/web app called PropelPixel. Functions similar to BeReal where users will be notified to take and share photos when traveling.

Ultimately, user accounts can be linked to a crypto-wallet where they can earn tokens when using public transit, carpool, walk, etc. The goal of this app is to incentivize users to use more eco-friendly forms of transportations.
